# 论文实证结果摘要

> 生成时间：2026-03-22
> 数据来源：CSMAR 数据库，A 股上市公司（剔除金融行业）
> 样本区间：2003–2024 年
> 方法：OLS + 行业/年份固定效应 + 公司层面聚类稳健标准误
>
> **本轮严格性修正**：
> 1. 原始薪酬数据沿用源数据既有缩尾口径，`lnCEOpay` 与 `Overpay` 不再二次缩尾
> 2. 中介效应改为“原始 Sobel + 分层 BH-FDR + 公司层面 cluster bootstrap”三层报告
> 3. 分组中介明确定位为探索性分析，不再以“做出显著组”为目标表述
> 4. 管理层权力 PCA 补充 KMO、Bartlett 球形检验和方差贡献率诊断
> 5. 机器学习部分保留为“基于机器学习的补充分析”，不作因果解释

## 一、主回归结果（表2，统一样本 N=44,831）

| 模型 | 因变量 | lnSubsidy | Power | R² |
|:---|:---|:---:|:---:|:---:|
| 模型(3) | Overpay | 0.0582*** (z=14.32) | — | 0.0372 |
| 模型(4) | Overpay | 0.0576*** (z=14.11) | 0.0062 (z=1.24) | 0.0374 |
| 模型(5) | Power | 0.0973*** (z=11.27) | — | 0.1427 |

**核心发现**：

1. **假设 H1 成立**：政府补助（lnSubsidy）显著正向影响高管超额薪酬。
2. **全样本中介效应不成立**：
   - 路径 a：lnSubsidy → Power，系数 0.0973***；
   - 路径 b：Power → Overpay，系数 0.0062，不显著；
   - Sobel p = 0.2168；
   - cluster bootstrap（300 次）经验 p = 0.1733，95% CI = [-0.000291, 0.001701]；
   - 间接效应占总效应比例约 1.04%。

结论：全样本层面不存在稳健中介效应证据。

## 二、分组与机制结果

### 2.1 分产权性质（表4）

| | 国有企业 (N=15,623) | 私营企业 (N=25,145) |
|:---|:---:|:---:|
| lnSubsidy (模型3) | 0.0437*** | 0.0723*** |
| lnSubsidy (模型4) | 0.0431*** | 0.0725*** |
| Power (模型4) | 0.0139 | -0.0014 |

**发现**：政府补助对私营企业超额薪酬的推动作用更强，但国有与私营直分样本下均未形成显著中介效应。

### 2.2 分行业管制（表5）

| | 管制行业 (N=7,121) | 非管制行业 (N=37,710) |
|:---|:---:|:---:|
| lnSubsidy (模型3) | 0.0260*** | 0.0662*** |

**发现**：管制行业中的补助效应显著低于非管制行业，行业监管表现出明显约束作用。

### 2.3 央企 vs 地方国企（表6）

| | 央企 (N=3,605) | 地方国企 (N=7,901) |
|:---|:---:|:---:|
| lnSubsidy (模型3) | 0.0205* | 0.0438*** |
| Power (模型4) | 0.0284* | 0.0262** |
| 原始 Sobel | 不显著 | p = 0.0837 |
| FDR 校正后 | 不显著 | 不显著 |
| Bootstrap 95%CI | — | [-0.000016, 0.003164] |

**发现**：

- 央企中补助效应仅边际显著，经济量值显著低于地方国企；
- 地方国企样本存在原始 Sobel 的边际证据，但经 FDR 校正与 bootstrap 复核后，尚不能视为稳健中介证据。

### 2.4 分组中介效应汇总（附表A）

分组中介采用两层口径：

- **产权直分**：国有、私营、央企、地方国企
- **补充分组**：东部、中西部、2003–2012、2013–2024、国有×东部、地方国企×东部、地方国企×后期

严格口径下，分组结果应结合原始 Sobel、层内 BH-FDR 和公司层面 cluster bootstrap 共同判断：

| 分组 | 原始 Sobel p | FDR p | Bootstrap 95%CI | 严格口径 |
|:---|:---:|:---:|:---:|:---|
| 东部 | 0.0204 | 0.0714 | [0.000353, 0.003077] | 探索性稳健支持 |
| 2003–2012 | 0.0153 | 0.0714 | [0.000341, 0.002819] | 探索性稳健支持 |
| 国有×东部 | 0.0503 | 0.0880 | [0.000278, 0.003445] | 探索性稳健支持 |
| 地方国企×东部 | 0.0372 | 0.0867 | [0.000514, 0.007701] | 探索性稳健支持 |
| 地方国企 | 0.0837 | 0.3347 | [-0.000016, 0.003164] | 仅原始 Sobel 支持 |
| 地方国企×后期 | 0.0902 | 0.1263 | [-0.000073, 0.003569] | 仅原始 Sobel 支持 |

解释：

- 若只看原始 Sobel，东部、2003–2012、地方国企×东部为显著组；
- 若加入层内 FDR 校正，这些结果降为 10% 水平的边际显著；
- 若进一步要求 bootstrap 95%CI 不含 0，则 **东部、2003–2012、国有×东部、地方国企×东部** 保留为“探索性稳健支持”；
- 产权直分层面没有任何一组达到稳健中介结论。

因此，分组中介应表述为：**存在若干探索性支持的制度情境，但不能将其上升为全样本普遍成立的中介机制。**

## 三、PCA 诊断结果

管理层权力综合指标基于 `Tenure`、`Dual`、`Boardsize`、`Insider`、`Mgshder` 五个指标构建，PCA 可用样本量为 44,965。

- **KMO = 0.5649**
- **Bartlett 球形检验**：Chi² = 19929.35，df = 10，p < 0.001
- **第一主成分方差贡献率 = 34.57%**
- 前两主成分累计方差贡献率 = 57.19%，前三主成分累计方差贡献率 = 75.84%

结论：管理层权力指标的 PCA 提取在统计上可以成立，但 KMO 不高，论文表述宜写为“通过基本适用性检验”。

## 五、基于机器学习的补充分析

### 5.1 Lasso 回归

- 最优正则化参数 α = 0.004357
- 样本内拟合 R² = 0.0401
- 保留变量共 10 个，其中包括 `lnSubsidy`、`Lever`、`Top1`、`Boardsize`、`Dual`、`Mgshder`、`IsSOE`、`Industry`、`Tenure`、`Roa`

### 5.2 随机森林

- 回归预测：R² = 0.1421，RMSE = 0.5237，MAE = 0.4033
- 5 折交叉验证 R² = 0.0694 ± 0.0182
- 分类预测准确率 = 0.6268
- 回归重要性最高的变量为 **Roa**，`lnSubsidy` 排名第二

### 5.3 XGBoost + SHAP

- 回归预测：R² = 0.2012，RMSE = 0.5054
- 5 折交叉验证 R² = 0.0723 ± 0.0278
- SHAP 重要性排名第一的变量为 **lnSubsidy**

### 5.4 决策树与 K-Means

- 决策树分类准确率 = 0.5580
- K-Means 最优聚类数 K = 2

### 5.5 模型性能对比

| 模型 | R² | RMSE | MAE |
|:---|:---:|:---:|:---:|
| Lasso 回归 | 0.0362 | 0.5551 | 0.4281 |
| 随机森林 | 0.1421 | 0.5237 | 0.4033 |
| XGBoost | 0.2012 | 0.5054 | 0.3900 |

解释：机器学习结果从预测角度支持 `lnSubsidy` 的重要性，但不改变主回归“全样本无稳健中介效应”的结论。

## 六、写作建议

论文正文可概括为：

**“全样本结果表明，财政补贴显著提高高管超额薪酬，但管理层权力并未形成稳健中介效应。进一步的分组检验显示，在东部、2003–2012、国有×东部和地方国企×东部样本中，原始 Sobel、层内 FDR 和 cluster bootstrap 提供了探索性稳健支持，说明该机制可能具有制度与情境依赖性；但这些证据仍属于补充分组中的探索性发现，应谨慎解释。”**
