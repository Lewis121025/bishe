# 数据分析部分（论文第四、五章）

> **论文题目**：基于数据挖掘的上市公司财政补贴与高管超额薪酬研究
> **数据来源**：CSMAR 数据库，沪深 A 股上市公司（剔除金融行业）
> **样本区间**：2003–2024 年
> **估计方法**：OLS + 行业固定效应 + 年份固定效应 + 公司层面聚类稳健标准误
> **样本处理**：原始薪酬数据沿用源数据既有缩尾口径，`lnCEOpay` 与 `Overpay` 不再二次缩尾；其余连续变量在 1% 和 99% 分位数处进行 Winsorize

---

## 4. 研究假设

**假设 H1（主效应假设）** 政府补助（lnSubsidy）对高管超额薪酬（Overpay）具有显著正向影响，即财政补贴越多，高管获得超额薪酬越高。

**假设 H2（中介效应假设）** 管理层权力（Power）在政府补助与高管超额薪酬之间发挥中介作用，即政府补助通过增强管理层权力间接提升高管超额薪酬。

---

### 5.3 全样本中介效应与 PCA 适用性

全样本中介效应检验结果见表 3。首先，lnSubsidy 对 Overpay 的总效应显著为正（0.0582，p < 0.01）；其次，lnSubsidy 对 Power 的路径 a 显著为正（0.0973，p < 0.01）；但在加入 Power 后，Power 对 Overpay 的路径 b 仅为 0.0062，统计上不显著（p = 0.2140）。原始 Sobel 检验得到 Z = 1.2352，p = 0.2168，间接效应占总效应比例仅为 1.04%。进一步采用公司层面 cluster bootstrap（300 次）后，间接效应经验 p = 0.1733，95% 置信区间为 [-0.000291, 0.001701]，同样跨越 0。

因此，全样本下**不存在稳健中介效应**。从结果解释看，政府补助确实能够提升管理层权力，但在覆盖全部 A 股上市公司的大样本条件下，这种权力变化并未普遍转化为高管超额薪酬。这意味着 H2 不能在全样本层面成立。

### 5.6 基于机器学习的补充分析

计量经济学方法主要用于关联与机制检验，而机器学习方法更擅长捕捉非线性关系和高维交互效应。二者目标不同，但可以相互补充。为从预测视角交叉验证计量回归结果，本文进一步使用 Lasso、随机森林、XGBoost、决策树和 K-Means 聚类开展补充分析。

#### 5.6.1 Lasso 回归——变量筛选

Lasso 采用 5 折交叉验证确定最优正则化参数，得到 α = 0.004357，样本内拟合 R² = 0.0401。最终保留的 10 个变量包括 `lnSubsidy`、`Lever`、`Top1`、`Boardsize`、`Dual`、`Mgshder`、`IsSOE`、`Industry`、`Tenure` 和 `Roa`。`lnSubsidy` 被稳定保留，说明其对超额薪酬具有稳健解释力。

#### 5.6.2 随机森林——特征重要性与预测

随机森林回归的样本外预测结果为：R² = 0.1421，RMSE = 0.5237，MAE = 0.4033；5 折交叉验证 R² = 0.0694 ± 0.0182。分类模型用于识别“是否存在超额薪酬”，准确率为 0.6268。回归特征重要性排序中，`Roa` 位列第一，`lnSubsidy` 位列第二，说明政府补助在非线性模型中依然具有重要作用。

#### 5.6.3 XGBoost + SHAP 可解释性分析

XGBoost 的样本外预测表现优于随机森林：R² = 0.2012，RMSE = 0.5054，5 折交叉验证 R² = 0.0723 ± 0.0278。SHAP 重要性排序显示，`lnSubsidy` 为第一重要变量，且 SHAP 值整体为正，说明政府补助的提高显著抬升了超额薪酬的预测值。这一结果与计量回归的主结论高度一致。

#### 5.6.4 决策树与 K-Means 企业画像

决策树分类准确率为 0.5580，虽然整体精度不高，但可视化规则显示“高补助 + 特定治理结构”是超额薪酬的重要风险路径。K-Means 聚类的最优聚类数为 K = 2，高补助聚类同时表现为更高杠杆、更高平均薪酬和更高国有企业占比，这与前文制度异质性分析的方向一致。

#### 5.6.5 模型性能对比

**表 8 机器学习模型预测性能对比**

## 6. 研究结论

本文基于 2003–2024 年沪深 A 股上市公司面板数据，综合运用计量经济学方法与机器学习技术，系统考察了政府补助对高管超额薪酬的影响及其内在机制。主要结论如下：

**（1）政府补助显著推高高管超额薪酬，H1 得到支持。** 在控制公司业绩、财务杠杆、股权集中度、区域特征以及行业和年份固定效应后，lnSubsidy 的系数在全样本中稳定为正且高度显著。

**（2）管理层权力的中介效应在全样本下不成立。** 虽然政府补助显著提升管理层权力，但 Power 对 Overpay 的直接作用不显著，原始 Sobel 与 cluster bootstrap 均不支持全样本中介，因此 H2 不能在全样本层面成立。

**（3）中介机制具有明显的情境依赖性，但相关证据属于探索性支持。** 在东部样本、2003–2012 样本、国有×东部样本以及地方国企×东部样本中，原始 Sobel、层内 FDR 和 cluster bootstrap 提供了一致的探索性稳健支持；地方国企样本与地方国企×后期样本仅保留原始 Sobel 的边际证据。这说明管理层权力的中介作用主要出现在特定制度环境和样本时期，而非普遍存在于全部上市公司。

**（4）产权、行业监管与国企层级会显著影响补助效应大小。** 私营企业中补助效应明显强于国有企业；非管制行业强于管制行业；地方国企强于央企。这表明外部监管和内部治理约束会改变财政补贴向高管超额薪酬转化的强度。

**（5）机器学习分析从预测视角验证了计量结论。** Lasso 保留了 `lnSubsidy` 等核心变量，XGBoost 的 SHAP 分析更将 `lnSubsidy` 排名为最重要特征，说明政府补助在非线性模型中同样是高管超额薪酬的重要驱动因素。

**实践启示**：财政部门和监管机构应重点关注补贴资金在东部地区及部分地方国企治理环境中的资源配置问题；企业层面则应强化董事会治理和薪酬约束机制，防止补贴资源被管理层内部化。对于分组中介结果，政策含义应以“探索性情境证据”而非“普遍机制”来表述。

**研究局限与展望**：本文尚未引入工具变量或动态面板处理潜在内生性问题，PCA 指标的 KMO 水平也仅属基本可接受。后续研究可进一步细化管理层权力度量，并结合文本数据与机器学习方法开展更高维度的机制识别。
