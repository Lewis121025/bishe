# 数据分析部分（论文第四、五章）

> **论文题目**：基于数据挖掘的上市公司财政补贴与高管超额薪酬研究
> **数据来源**：CSMAR 数据库，沪深 A 股上市公司（剔除金融行业）
> **样本区间**：2003–2024 年
> **估计方法**：OLS + 行业固定效应 + 年份固定效应 + 公司层面聚类稳健标准误
> **样本处理**：连续变量在 1% 和 99% 分位数处进行 Winsorize 缩尾；超额薪酬（Overpay）作为期望薪酬模型残差，同样在 1%/99% 分位数处进行 Winsorize 缩尾

---

## 4. 研究假设

**假设 H1（主效应假设）** 政府补助（lnSubsidy）对高管超额薪酬（Overpay）具有显著正向影响，即财政补贴越多，高管获得超额薪酬越高。根据代理理论，在股东与经理层信息不对称的条件下，财政补贴增大了企业可支配资源规模，当公司治理机制不完善时，高管可能借助这一外部资源拉升自身薪酬水平（Bebchuk & Fried，2003）。

**假设 H2（中介效应假设）** 管理层权力（Power）在政府补助与高管超额薪酬之间发挥显著中介作用，即政府补助通过增强管理层权力间接提升高管超额薪酬。政府补贴增大了管理层可支配的资源规模，强化了管理层在薪酬谈判中的议价能力，进而推高超额薪酬。

### 5.3 中介效应检验

表 3 报告了基于 Baron & Kenny（1986）三步法及 Sobel 检验的管理层权力中介效应结果。

第（1）步：lnSubsidy 对 Overpay 的总效应显著为正（0.0552，t = 14.56），路径 c 成立；

第（2）步：lnSubsidy 对 Power 显著正向影响（0.0974，t = 11.28），路径 a 成立；

第（3）步：加入 Power 后，Power 对 Overpay 的系数为 0.0066（t = 1.37），统计上不显著，路径 b **不成立**；lnSubsidy 直接效应系数从 0.0552 微降至 0.0545，几乎无变化。

Sobel 检验显示间接效应为 a × b = 0.0974 × 0.0066 ≈ 0.000643，**完全不显著**，中介效应占总效应比例仅约 1.16%。

**表 3 管理层权力中介效应检验**

|:---|:---:|:---:|:---:|

### 5.5 稳健性检验

为确保结论可靠，本文进行了六项稳健性检验（结果见表 7）：

1. **替换被解释变量**：以高管薪酬对数（lnCEOpay）替代超额薪酬（Overpay），lnSubsidy 系数为 0.1319（t = 32.94，p < 0.01），结论稳健。
2. **缩小样本期**：仅保留 2010–2020 年数据，lnSubsidy 系数为 0.0536（t = 12.48，p < 0.01），结论不变。
3. **剔除极端补助值**：去除政府补助 5%/95% 分位数以外的观测值，lnSubsidy 系数为 0.0577（t = 12.91，p < 0.01），结论稳健。
4. **仅保留制造业样本**：lnSubsidy 系数为 0.0709（t = 14.83，p < 0.01），结论稳健。
5. **替换解释变量**：以 ln(1 + 补助) 替代 ln(补助)，系数为 0.0509（t = 14.77，p < 0.01），结论不变。
6. **滞后一期补助**：使用 t-1 期政府补助替代当期补助，系数为 0.0508（t = 13.92，p < 0.01），在一定程度上缓解了反向因果的担忧。

### 5.6 基于机器学习的补充分析

计量经济学方法侧重于因果推断与假设检验，而机器学习方法在捕捉非线性关系与高维变量交互效应方面具有独特优势。两类方法的研究目标不同，互为补充而非替代。为从预测与模式发现的角度进一步验证计量回归的核心结论，本文采用 Lasso 回归、随机森林、XGBoost、决策树及 K-Means 聚类等多种机器学习方法进行补充分析。

#### 5.6.1 Lasso 回归——变量筛选

采用 5 折交叉验证选择最优正则化参数 α，通过 Lasso 回归对 14 个候选特征进行变量筛选。Lasso 回归的核心目的是通过 L₁ 正则化惩罚项，将不重要变量的系数压缩至零，从而实现自动变量筛选。结果表明：lnSubsidy、Roa、Lever、Top1 等核心变量被保留，验证了 OLS 回归模型中变量选取的合理性。被剔除的变量（如 IsSOE、Zone 等）在经济意义上属于分组变量而非直接解释变量，其系数被压缩至零符合预期。

#### 5.6.2 随机森林——特征重要性与预测

随机森林回归模型（n_estimators = 200，max_depth = 10）用于预测超额薪酬数值，同时通过特征重要性排序揭示各变量的相对贡献。主要结果如下：

- 回归预测：R² ≈ 0.22，RMSE ≈ 0.59，5 折交叉验证 R² 稳定；
- 分类预测（是否存在超额薪酬）：准确率约 66%；
- 特征重要性排名：lnSubsidy 位列前列，与 OLS 回归的核心结论一致。

#### 5.6.3 XGBoost + SHAP 可解释性分析

XGBoost 梯度提升模型（n_estimators = 300，max_depth = 6，learning_rate = 0.05）在预测精度上优于随机森林，并通过 SHAP（SHapley Additive exPlanations）值分析实现模型的可解释性。

- XGBoost 回归 R² 高于随机森林，5 折交叉验证结果稳定；
- **SHAP 特征重要性排名**：lnSubsidy 位列重要性前列，其 SHAP 值整体为正，即政府补助增加倾向于推高超额薪酬，与 OLS 回归结论高度一致；
- **SHAP 依赖图**显示 lnSubsidy 与超额薪酬之间呈现正向关系，但存在轻微的非线性特征——当补助规模达到较高水平时，边际影响有所减弱。

#### 5.6.4 决策树——超额薪酬风险路径

采用浅层决策树（max_depth = 4）进行分类建模，直观展示哪些条件组合最可能导致高管超额薪酬为正。决策树揭示了"高政府补助 + 低财务杠杆 + 低股权集中度"的典型高风险路径，为监管机构识别补贴向超额薪酬转化的高风险企业提供了实践参考。

#### 5.6.5 K-Means 聚类——企业画像

采用 K-Means 聚类算法对企业进行分群画像分析。基于 lnSubsidy、Overpay、Roa、Lever、Top1、lnSale 六个变量进行聚类，通过轮廓系数确定最优聚类数 K。聚类结果揭示了不同类型企业群的差异化特征：高补助-高超额薪酬群 vs. 低补助-低超额薪酬群的对比印证了主回归发现，不同聚类群内国有企业占比的差异也与产权异质性分析的结论一致。

#### 5.6.6 模型预测性能对比

表 8 报告了三种机器学习模型在超额薪酬预测任务上的性能对比。需要指出的是，机器学习模型的预测性能指标（R²、RMSE、MAE）衡量的是模型对样本外数据的预测能力，与 OLS 回归的因果推断目标不同，二者不可直接比较。机器学习分析的价值在于：（1）通过特征重要性排序与 SHAP 值分析，从预测视角独立验证了 lnSubsidy 对超额薪酬的正向关联；（2）揭示了潜在的非线性效应和变量交互模式，为计量回归的线性假定提供了补充检验。

三种机器学习模型的特征重要性排序均将 lnSubsidy 列为关键预测变量之一，且 SHAP 分析一致显示其正向效应，与计量回归的核心结论相互印证，从方法论互补的角度增强了实证结果的可信度。

## 6. 研究结论

本文基于 2003–2024 年沪深 A 股上市公司面板数据，综合运用计量经济学方法和机器学习技术，系统考察了政府补助对高管超额薪酬的影响及其内在机制。实证结果表明：

**（1）政府补助对高管超额薪酬具有显著正向影响（H1 得到支持）。** 在控制业绩、财务杠杆、股权结构、区域特征以及行业和年份固定效应后，lnSubsidy 系数在 1% 水平下显著为正（0.0552），经济意义上，政府补助每增加 1%，超额薪酬增加约 0.055 个单位。该效应在六项稳健性检验中保持一致。

**（2）管理层权力的中介效应在全样本下不成立（H2 在全样本下不支持）。** 虽然政府补助对管理层权力有显著正向影响（路径 a 成立，系数 0.0974），但管理层权力对超额薪酬的影响微弱且统计不显著（路径 b 不成立），Sobel 检验完全不显著。该结果可能源于样本扩展至全部 A 股后，管理层权力理论所依赖的所有者缺位等制度前提在非国有企业中并不适用。

**（3）补贴效应存在显著的产权异质性。** 非国有企业中补贴效应显著强于国有企业（0.0734 vs. 0.0416），表明国有企业的薪酬管控制度对补贴向超额薪酬的转化起到了一定的约束作用。

**（4）行业监管和国企层级对补贴效应有显著的调节作用。** 管制行业中补贴效应（0.0260）显著低于非管制行业（0.0625），说明行业监管对补贴效应具有实质性约束；央企（0.0202）与地方国企（0.0432）的效应差距明显。值得注意的是，管理层权力（Power）在央企（p < 0.1）和地方国企（p < 0.05）子样本中均显著为正，表明管理层权力的中介机制具有**制度依赖性**，在国有产权结构下更为突出，这为未来研究提供了有价值的线索。

**（5）机器学习方法从预测与模式发现的角度交叉验证了上述结论。** Lasso 回归保留了 lnSubsidy 等核心变量，验证了计量模型中变量选取的合理性；XGBoost 的 SHAP 分析将 lnSubsidy 列为重要预测特征之一，且 SHAP 值呈现正向效应，与 OLS 回归的因果推断结论一致；决策树揭示了"高补助 + 低杠杆 + 低股权集中度"的超额薪酬高风险路径；K-Means 聚类分析的企业画像结果与产权异质性分析相吻合。需要强调的是，机器学习方法侧重预测与关联分析，不具备计量经济学方法的因果识别能力，二者互为补充而非替代。

**理论贡献：** 本文在管理者权力理论和代理理论的框架下，通过大样本实证检验，揭示了政府补助对高管超额薪酬的正向影响及其在不同产权结构、行业管制和国企层级下的差异化表现。研究发现管理层权力的中介作用具有制度依赖性，在国有产权结构（尤其是央企和地方国企）下更为明显，丰富了对财政补贴治理效应的理论理解。

**实践启示：** 财政部门应加强对补贴资金使用的监督与绩效评估，尤其关注非国有企业和非管制行业中补贴资金可能被高管攫取的风险；监管机构可借鉴本文机器学习方法，构建基于多维特征的超额薪酬风险预警模型，识别高风险企业；企业自身应完善内部控制与薪酬治理机制，确保补贴资金合理使用。

**研究局限与展望：** 本文虽控制了行业和年份固定效应并采用了多种稳健性检验，但仍存在以下不足：（1）未引入工具变量法处理潜在的内生性问题，OLS 估计可能存在遗漏变量偏差；（2）管理层权力的度量依赖于 PCA 综合指标，未来可尝试更精细的权力维度分解；（3）未纳入在职消费等隐性薪酬维度，可能低估了补贴对高管实际收益的影响。未来研究可进一步引入动态面板模型或工具变量法以强化因果识别，并尝试将文本分析方法与机器学习相结合，构建更为全面的超额薪酬风险评估体系。
