# 回归模型设定

## 一、变量定义

### 被解释变量
- **lnCEOpay**：高管前三名薪酬总额取对数；原始薪酬数据沿用源数据既有缩尾口径，本文不再额外对 `lnCEOpay` 二次缩尾
- **Overpay**（超额薪酬）：模型(1)回归残差，反映实际薪酬超出“合理水平”的部分；`Overpay` 直接使用残差值，不再进行二次 Winsorize

### 核心解释变量
- **lnSubsidy**（政府补助）：仅对正值政府补助取对数
- **lnSubsidy1p**：稳健性替代变量，记为 `ln(1 + 补助)`

### 中介变量
- **Power**（管理层权力）：由任期 `Tenure`、两职合一 `Dual`、董事会规模 `Boardsize`、内部董事比例 `Insider`、管理层持股比例 `Mgshder` 五个指标经 PCA 提取第一主成分
- PCA 诊断结果：
  - KMO = 0.5649
  - Bartlett 球形检验 p < 0.001
  - 第一主成分方差贡献率 = 34.57%

### 控制变量
- **Roa**：资产收益率 = 净利润 / 总资产
- **Lever**：财务杠杆 = 总负债 / 总资产
- **Top1**：第一大股东持股比例(%)
- **Zone**：区域虚拟变量，东部 = 0，中西部 = 1

### 分组变量
- **IsSOE**：国有企业
- **IsPrivate**：私营企业（`Ownership = 私营企业`）
- **IsCentralSOE**：央企 / 地方国企（实控人代码 `2100=央企`，`2120=地方国企`）
- **RegulatedIndustry**：管制行业（行业大类 `B/D/G/I/N`）

## 二、模型设定

### 模型(1)：期望薪酬估计

$$lnCEOpay_{i,t} = \alpha_0 + \alpha_1 lnSale_{i,t} + \alpha_2 Roa_{i,t} + \alpha_3 IA_{i,t} + \alpha_4 Zone_i + \sum Industry + \sum Year + \varepsilon_{i,t}$$

其中：

- `lnSale = ln(营业收入)`，非正营业收入在取对数前置缺失
- `IA = 无形资产 / 总资产`
- `ΣIndustry`、`ΣYear` 分别表示行业和年份固定效应

### 模型(2)：超额薪酬计算

$$Overpay_{i,t} = lnCEOpay_{i,t} - \widehat{lnCEOpay_{i,t}}$$

即模型(1)回归残差。由于原始薪酬数据已沿用既有缩尾口径，`Overpay` 不再二次缩尾。

### 模型(3)：基准回归（H1）

$$Overpay_{i,t} = \beta_0 + \beta_1 lnSubsidy_{i,t} + \beta_2 Roa_{i,t} + \beta_3 Lever_{i,t} + \beta_4 Top1_{i,t} + \beta_5 Zone_i + \sum Industry + \sum Year + \varepsilon_{i,t}$$

### 模型(4)：加入中介变量

$$Overpay_{i,t} = \gamma_0 + \gamma_1 lnSubsidy_{i,t} + \gamma_2 Power_{i,t} + \gamma_3 Roa_{i,t} + \gamma_4 Lever_{i,t} + \gamma_5 Top1_{i,t} + \gamma_6 Zone_i + \sum Industry + \sum Year + \varepsilon_{i,t}$$

### 模型(5)：中介路径检验

$$Power_{i,t} = \delta_0 + \delta_1 lnSubsidy_{i,t} + \delta_2 Roa_{i,t} + \delta_3 Lever_{i,t} + \delta_4 Top1_{i,t} + \delta_5 Zone_i + \sum Industry + \sum Year + \varepsilon_{i,t}$$

## 三、中介效应检验框架

### 1. 全样本检验

全样本中介效应采用三层报告：

1. **Baron & Kenny（1986）三步法**
2. **Sobel 检验**
3. **公司层面 cluster bootstrap（300 次）**

步骤如下：

1. 模型(3)检验 `lnSubsidy → Overpay` 的总效应 `c`
2. 模型(5)检验 `lnSubsidy → Power` 的路径 `a`
3. 模型(4)同时检验 `Power → Overpay` 的路径 `b` 与 `lnSubsidy → Overpay` 的直接效应 `c'`

Sobel 统计量为：

$$Z_{sobel} = \frac{a \times b}{\sqrt{b^2 \cdot se_a^2 + a^2 \cdot se_b^2}}$$

Bootstrap 采用**公司层面 cluster resampling**，以间接效应 `a×b` 的 95% 置信区间是否跨 0 作为补充判定标准。

### 2. 分组中介检验

分组中介效应采用与全样本完全一致的模型(3)/(4)/(5)，并新增两层严谨性处理：

1. **原始 Sobel p 值**
2. **层内 BH-FDR 校正**
3. **对原始 Sobel p < 0.10 的组执行公司层面 cluster bootstrap（300 次）**

原始 Sobel 的判定规则为：

- `Sobel p < 0.05`：存在中介效应
- `0.05 ≤ Sobel p < 0.10`：边际中介效应
- `Sobel p ≥ 0.10`：不存在中介效应

FDR 校正在以下两个层级内分别进行：

- **产权直分**：国有、私营、央企、地方国企
- **补充分组**：东部、中西部、2003–2012、2013–2024、国有×东部、地方国企×东部、地方国企×后期

### 3. 严格口径的结论规则

为避免将事后分组检验解释过满，正文中的“严格口径”按以下规则表述：

- **校正后稳健支持**：FDR p < 0.05 且 bootstrap 95%CI 不含 0
- **探索性稳健支持**：FDR p < 0.10 且 bootstrap 95%CI 不含 0
- **仅原始 Sobel 支持**：原始 Sobel 达到 10% 水平，但 FDR 或 bootstrap 未支持
- **无稳健证据**：原始 Sobel、FDR 与 bootstrap 均不支持

## 四、估计方法

- **回归方法**：OLS
- **标准误**：公司层面聚类稳健标准误
- **固定效应**：行业固定效应 + 年份固定效应
- **样本处理**：
  - `lnCEOpay`、`Overpay` 不做二次缩尾
  - 其他连续变量如 `lnSubsidy`、`lnSale`、`Roa`、`IA`、`Lever`、`Top1` 在 1% 和 99% 分位数处进行 Winsorize

## 五、已报告的主要分组

| 分组维度 | 分组方式 | 用途 |
|:---|:---|:---|
| 产权直分 | 国有 vs 私营 | 检验产权性质差异 |
| 行业管制 | 管制行业 vs 非管制行业 | 检验行业监管约束 |
| 国企层级 | 央企 vs 地方国企 | 检验不同监管层级差异 |
| 中介补充分组 | 东部、2003–2012、国有×东部、地方国企×东部等 | 报告探索性制度情境 |
