# 回归模型设定

## 一、变量定义

### 被解释变量
- **Overpay**（超额薪酬）：模型(1)回归残差，反映实际薪酬超出"合理水平"的部分

### 核心解释变量
- **lnSubsidy**（政府补助）：ln(政府补助金额)

### 中介变量
- **Power**（管理层权力）：由任期(Tenure)、两职合一(Dual)、董事会规模(Boardsize)、内部董事比例(Insider)、管理层持股比例(Mgshder) 五个指标经PCA提取第一主成分

### 控制变量
- **Roa**：资产收益率 = 净利润/总资产
- **Lever**：财务杠杆 = 总负债/总资产
- **Top1**：第一大股东持股比例(%)
- **Zone**：区域虚拟变量，东部=0，中西部=1

---

## 二、模型设定

### 模型(1)：期望薪酬估计

$$lnCEOpay_{i,t} = \alpha_0 + \alpha_1 lnSale_{i,t} + \alpha_2 Roa_{i,t} + \alpha_3 IA_{i,t} + \alpha_4 Zone_i + \sum Industry + \varepsilon_{i,t}$$

其中：
- lnCEOpay = ln(高管前三名薪酬总额)
- lnSale = ln(营业收入)
- IA = 无形资产/总资产
- ΣIndustry = 行业固定效应

**目的**：估计"正常应得"的薪酬水平

### 模型(2)：超额薪酬计算

$$Overpay_{i,t} = lnCEOpay_{i,t} - \widehat{lnCEOpay_{i,t}}$$

即模型(1)的回归残差，正值=超额，负值=低于预期。

---

### 模型(3)：基准回归（假设H1检验）

$$Overpay_{i,t} = \beta_0 + \beta_1 lnSubsidy_{i,t} + \beta_2 Roa_{i,t} + \beta_3 Lever_{i,t} + \beta_4 Top1_{i,t} + \beta_5 Zone_i + \sum Industry + \sum Year + \varepsilon_{i,t}$$

**检验**：β₁ 是否显著为正？若 β₁ > 0 且显著 → H1成立（政府补助增加超额薪酬）

### 模型(4)：加入中介变量

$$Overpay_{i,t} = \gamma_0 + \gamma_1 lnSubsidy_{i,t} + \gamma_2 Power_{i,t} + \gamma_3 Roa_{i,t} + \gamma_4 Lever_{i,t} + \gamma_5 Top1_{i,t} + \gamma_6 Zone_i + \sum Industry + \sum Year + \varepsilon_{i,t}$$

**检验（对比模型3）**：
- 若 γ₁ < β₁ 且 γ₂ 显著 → Power 有中介效应
- 若 γ₁ 不显著 → 完全中介
- 若 γ₁ 仍显著但变小 → 部分中介

### 模型(5)：中介路径检验

$$Power_{i,t} = \delta_0 + \delta_1 lnSubsidy_{i,t} + \delta_2 Roa_{i,t} + \delta_3 Lever_{i,t} + \delta_4 Top1_{i,t} + \delta_5 Zone_i + \sum Industry + \sum Year + \varepsilon_{i,t}$$

**检验**：δ₁ 是否显著？即政府补助是否影响管理层权力

---

## 三、中介效应检验框架（Baron & Kenny, 1986）

```
步骤1：模型(3)检验 lnSubsidy → Overpay 的总效应（β₁）
步骤2：模型(5)检验 lnSubsidy → Power 的路径a（δ₁）
步骤3：模型(4)检验加入Power后：
        - lnSubsidy → Overpay 的直接效应（γ₁）
        - Power → Overpay 的路径b（γ₂）
```

中介效应成立条件：
1. β₁ 显著（总效应存在）
2. δ₁ 显著（自变量影响中介变量）
3. γ₂ 显著（中介变量影响因变量）
4. γ₁ < β₁（直接效应减弱）

**补充检验**：Sobel检验
$$Z_{sobel} = \frac{\delta_1 \times \gamma_2}{\sqrt{\gamma_2^2 \cdot se_{\delta_1}^2 + \delta_1^2 \cdot se_{\gamma_2}^2}}$$

---

## 四、估计方法

- **回归方法**：OLS（普通最小二乘法）
- **标准误**：公司层面聚类稳健标准误（Cluster-Robust SE, clustered at firm level）
- **固定效应**：行业固定效应 + 年份固定效应
- **样本处理**：连续变量在1%和99%分位数处进行Winsorize缩尾

---

## 五、异质性分析（分组回归）

在不同子样本上重复模型(3)和模型(4)：

| 分组维度 | 分组方式 | 目的 |
|:---|:---|:---|
| 产权性质 | 国有 vs 非国有 | 检验国有监管是否约束补助效应 |
| 行业管制 | 管制行业 vs 非管制行业 | 检验行业监管是否约束补助效应 |
| 国企层级 | 央企 vs 地方国企 | 检验不同监管力度下的差异 |

## 六、稳健性检验

| 编号 | 方法 | 修改内容 |
|:---|:---|:---|
| (1) | 替换因变量 | 用 lnCEOpay 替代 Overpay |
| (2) | 缩小样本期 | 仅保留 2010-2020 年 |
| (3) | 剔除极端值 | 去掉补助 5%/95% 分位的观测 |
| (4) | 子样本 | 仅保留制造业 |
| (5) | 替换解释变量 | 用 ln(1+补助) 替代 ln(补助) |
| (6) | 滞后变量 | 用 t-1 期补助替代当期补助 |
