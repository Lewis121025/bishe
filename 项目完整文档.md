## 2. 研究主题与核心问题

论文题目：**基于数据挖掘的上市公司财政补贴与高管超额薪酬研究**

核心问题：

1. 财政补贴是否显著提高上市公司高管超额薪酬？
2. 管理层权力是否在“财政补贴 → 超额薪酬”之间发挥中介作用？
3. 该效应是否存在产权异质性（国有/私营）？
4. 该效应是否在管制行业、央地国企及其他补充分组中呈现机制差异？
5. 数据挖掘方法能否作为回归模型的补充验证，提高对超额薪酬的预测与解释能力？

---

### 3.1 计量经济学主线

## 4. 数据来源与样本口径

数据来源：CSMAR（国泰安）数据库。  
样本区间：2003-2024 年 A 股上市公司（剔除金融行业）。  
当前过滤后的基础样本量：**52,358** 个公司-年观测值。  
统一主回归与中介检验样本量：**44,831** 个公司-年观测值。

原始数据位于 `data/`，主要包括：

- 营业收入、净利润、总资产、无形资产、行业
- 政府补助（两文件拼接）
- 高管前三名薪酬总和
- 公司治理（两职合一、管理层持股、董事会规模、独董比例）
- 总经理任期
- 总负债
- 所在地与公司性质
- 第一大股东持股比例与实际控制人性质

## 5. 变量定义与构造口径

### 5.1 被解释变量

- `lnCEOpay`：高管前三名薪酬总和取对数，仅保留正值薪酬观测
- `Overpay`：超额薪酬，定义为实际薪酬对数减去期望薪酬对数，直接使用模型(1)残差值

### 5.2 核心解释变量

- `lnSubsidy`：政府补助对数（仅对正补助取对数，非正补助置缺失）
- `lnSubsidy1p`：稳健性替代变量，`ln(1 + 补助)`

### 5.3 中介变量

- `Power`：管理层权力综合指标  
  使用 `Tenure`、`Dual`、`Boardsize`、`Insider`、`Mgshder` 标准化后进行 PCA，取第一主成分

### 5.4 控制变量

- `Roa`：净利润 / 总资产
- `Lever`：总负债 / 总资产
- `Top1`：第一大股东持股比例
- `Zone`：区域虚拟变量（东部=0，中西部=1）
- 行业固定效应、年份固定效应

### 5.5 分组变量

- `IsSOE`：产权性质（国有/非国有广义识别）
- `IsPrivate`：产权直分中的私营企业识别变量
- `RegulatedIndustry`：管制行业（行业大类 `B/D/G/I/N`）
- `IsCentralSOE`：央企/地方国企（实控人代码 `2100=央企`，`2120=地方国企`）

## 6. 模型设定

### 6.1 期望薪酬模型（模型1）

`lnCEOpay = α₀ + α₁ lnSale + α₂ Roa + α₃ IA + α₄ Zone + ΣIndustry + ΣYear + ε`

> 模型残差即超额薪酬（Overpay）。原始薪酬变量已按既定口径缩尾，本轮修正后不再对残差进行二次 Winsorize；`lnSale` 非正值置缺失。

### 6.2 基准回归（模型3）

`Overpay = α₀ + α₁ lnSubsidy + Controls + ΣIndustry + ΣYear + ε`

### 6.3 中介检验（模型4、模型5）

- 模型4：  
`Overpay = α₀ + α₁ lnSubsidy + α₂ Power + Controls + ΣIndustry + ΣYear + ε`
- 模型5：  
`Power = α₀ + α₁ lnSubsidy + Controls + ΣIndustry + ΣYear + ε`

说明：

- 主回归标准误采用**公司层面聚类稳健标准误**
- 全样本中介效应采用 Baron & Kenny 逐步法 + Sobel 检验 + 公司层面 cluster bootstrap（300 次）
- 分组中介在“产权直分”“补充分组”两个层级内分别进行 BH-FDR 校正
- 对原始 Sobel `p<0.10` 的分组进一步实施公司层面 cluster bootstrap（300 次）

### 6.4 分组中介检验

第一层固定报告产权直分：

- 国有企业
- 私营企业
- 央企
- 地方国企

第二层补充分组固定报告：

- 东部
- 中西部
- 2003-2012
- 2013-2024
- 国有×东部
- 地方国企×东部
- 地方国企×后期

### 6.5 稳健性检验（6组）

1. 替换被解释变量为 `lnCEOpay`
2. 缩短样本期（2010-2020）
3. 剔除补贴极值（5%-95%）
4. 制造业子样本
5. 替换解释变量为 `lnSubsidy1p`
6. 使用滞后一期补贴 `lnSubsidy_l1`

## 10. 当前实证结果（2026-03-22 版）

> **本次修正重点**：删除 `Overpay` 残差二次缩尾；统一产权分组为“国有/私营”；补充分组中介检验；补齐管理层权力 PCA 诊断；新增 FDR 与 cluster bootstrap 作为中介严谨性复核；同步全部文档。

### 10.1 主回归与全样本中介

统一样本：`N = 44,831`，公司聚类数：`5,223`

| 模型 | 因变量 | lnSubsidy | Power | R² |
|:---|:---|:---:|:---:|:---:|
| 模型(3) | Overpay | 0.0582*** | — | 0.0372 |
| 模型(4) | Overpay | 0.0576*** | 0.0062 | 0.0374 |
| 模型(5) | Power | 0.0973*** | — | 0.1427 |

全样本 Sobel 检验结果：

- 间接效应：`0.000606`
- Sobel `z = 1.2352`
- Sobel `p = 0.2168`
- Bootstrap `p = 0.1733`
- Bootstrap 95% CI = `[-0.000291, 0.001701]`
- 中介占比：`1.04%`

结论：财政补贴显著提高高管超额薪酬，但全样本下管理层权力中介效应**不存在稳健证据**。

### 10.2 产权异质性与机制检验

| 分组 | 样本量 | lnSubsidy 系数 | Power 系数 | 结论 |
|:---|---:|:---:|:---:|:---|
| 国有企业 | 15,623 | 0.0437*** | 0.0139 | 补助效应显著，中介不显著 |
| 私营企业 | 25,145 | 0.0723*** | -0.0014 | 补助效应更强，中介不显著 |
| 管制行业 | 7,121 | 0.0260*** | 0.0017 | 监管约束更强 |
| 非管制行业 | 37,710 | 0.0662*** | 0.0073 | 补助效应更强 |
| 央企 | 3,605 | 0.0205* | 0.0284* | 补助效应较弱 |
| 地方国企 | 7,901 | 0.0438*** | 0.0262** | 存在更明显制度弹性 |

### 10.3 分组中介检验

| 分组 | 原始 Sobel p | FDR p | Bootstrap 95%CI | 严格口径 |
|:---|:---:|:---:|:---:|:---|
| 国有 | 0.1735 | 0.3471 | 未执行 | 无稳健证据 |
| 私营 | 0.8352 | 0.9306 | 未执行 | 无稳健证据 |
| 央企 | 0.9306 | 0.9306 | 未执行 | 无稳健证据 |
| 地方国企 | 0.0837 | 0.3347 | [-0.000016, 0.003164] | 仅原始Sobel支持 |
| 东部 | 0.0204 | 0.0714 | [0.000353, 0.003077] | 探索性稳健支持 |
| 中西部 | 0.2619 | 0.3055 | 未执行 | 无稳健证据 |
| 2003-2012 | 0.0153 | 0.0714 | [0.000341, 0.002819] | 探索性稳健支持 |
| 2013-2024 | 0.8837 | 0.8837 | 未执行 | 无稳健证据 |
| 国有×东部 | 0.0503 | 0.0880 | [0.000278, 0.003445] | 探索性稳健支持 |
| 地方国企×东部 | 0.0372 | 0.0867 | [0.000514, 0.007701] | 探索性稳健支持 |
| 地方国企×后期 | 0.0902 | 0.1263 | [-0.000073, 0.003569] | 仅原始Sobel支持 |

结论：全样本中介不成立。分组层面仅在**东部、2003-2012、国有×东部、地方国企×东部**样本中观察到经 FDR 与 bootstrap 复核后的**探索性稳健支持**；其余结果不宜作稳健中介结论。

### 10.4 PCA 诊断结果

基于 `Power` 实际建模样本 `N = 44,965`：

- KMO = `0.5649`
- Bartlett 球形检验：`χ² = 19929.3452`，`df = 10`，`p < 0.001`
- 第一主成分方差贡献率：`34.57%`
- 前两主成分累计贡献率：`57.19%`

解释：管理层权力综合指标通过基本适用性检验，Bartlett 检验显著，但 KMO 水平一般，因此论文中宜表述为“满足基本提取条件”。

### 10.5 稳健性检验

| 检验 | 系数 | R² |
|:---|:---:|:---:|
| (1) 替换因变量 `lnCEOpay` | 0.1365*** | 0.4234 |
| (2) 缩小期间 2010-2020 | 0.0551*** | 0.0353 |
| (3) 剔除极端补助值 | 0.0594*** | 0.0351 |
| (4) 仅制造业 | 0.0751*** | 0.0500 |
| (5) `ln(1+补助)` | 0.0539*** | 0.0375 |
| (6) 滞后一期补助 | 0.0530*** | 0.0364 |

6 组稳健性检验中，补贴系数均保持正向显著，主结论稳定。

### 10.6 基于机器学习的补充分析

Lasso、随机森林、XGBoost、决策树与 K-Means 均已重跑，核心结果如下：

- Lasso 最优惩罚参数 `alpha = 0.004357`，拟合 `R² = 0.0401`
- Lasso 保留变量：`lnSubsidy`、`Lever`、`Top1`、`Boardsize`、`Dual`、`Mgshder`、`IsSOE`、`Industry`、`Tenure`、`Roa`
- 随机森林：`R² = 0.1421`，`RMSE = 0.5237`，`MAE = 0.4033`，5 折交叉验证 `R² = 0.0694 ± 0.0182`
- 随机森林分类准确率：`0.6268`
- XGBoost：`R² = 0.2012`，`RMSE = 0.5054`，5 折交叉验证 `R² = 0.0723 ± 0.0278`
- 决策树分类准确率：`0.5580`
- K-Means 最优聚类数：`2`
- SHAP 显示 `lnSubsidy` 为最重要解释变量之一；随机森林中 `Roa` 贡献最高，`lnSubsidy` 位列前列

解释：机器学习模型在预测性能上优于线性基准模型，且对补助变量重要性的识别与回归结论一致，构成对计量结果的补充验证。
